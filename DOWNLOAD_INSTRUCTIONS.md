# 실행 파일 다운로드 방법

## 빠른 시작 (권장)

1. **GitHub 저장소에서 실행 파일 다운로드**

   - 저장소의 `자동검수/dist/` 폴더로 이동
   - `AI번역검수도구.exe` 파일을 다운로드
   - 파일을 더블클릭하여 실행

2. **Releases 페이지에서 다운로드** (가장 간단)
   - [Releases 페이지](https://github.com/tinklewhale/AI_L10N_check/releases)로 이동
   - 최신 버전의 `AI번역검수도구.exe` 다운로드
   - 파일을 더블클릭하여 실행

## 실행 파일 특징

- ✅ **Python 설치 불필요**: 독립 실행 파일
- ✅ **최적화됨**: 불필요한 패키지 제외로 파일 크기 최소화
- ✅ **빠른 다운로드**: 약 30-50% 작은 파일 크기
- ✅ **즉시 실행**: 다운로드 후 바로 사용 가능

## 시스템 요구사항

- Windows 10 이상
- 인터넷 연결 (API 호출용)
- Gemini API 키

## 문제 해결

### 실행 파일이 실행되지 않는 경우

1. **Windows Defender 경고**

   - "추가 정보" 클릭
   - "실행" 클릭
   - 또는 Windows Defender에서 예외 추가

2. **관리자 권한 필요**

   - 파일을 우클릭
   - "관리자 권한으로 실행" 선택

3. **바이러스 백신 프로그램 차단**
   - 바이러스 백신 프로그램에서 예외 추가
   - 실행 파일을 신뢰할 수 있는 위치로 이동

## 소스 코드에서 빌드하기

실행 파일을 직접 빌드하려면:

1. Python 3.8 이상 설치
2. 필요한 패키지 설치:
   ```bash
   cd 자동검수
   pip install -r requirements.txt
   ```
3. 최적화된 빌드 실행:
   ```bash
   BUILD_OPTIMIZED.bat
   ```

## 파일 크기 비교

- **기본 빌드**: 약 150-200 MB
- **최적화 빌드**: 약 80-120 MB (약 40% 감소)

최적화 빌드는 불필요한 패키지(matplotlib, scipy, jupyter 등)를 제외하여 파일 크기를 줄였습니다.
