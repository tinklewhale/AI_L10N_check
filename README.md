# AI 번역 검수 도구

번역 품질을 AI로 검수하는 GUI 애플리케이션입니다.

## 기능

- Gemini API를 사용한 번역 품질 검수
- 다양한 언어 지원 (한국어, 영어, 일본어, 중국어 등)
- 검수 결과를 4개 열로 분리 저장:
  - **AI 번역**: 개선된 번역 텍스트
  - **AI 검수 등급**: 등급 (양호/보통/크리티컬)
  - **AI 검수**: 등급 판단 이유
  - **일관성 체크**: 일관성 분석
- 실시간 진행률 표시
- 엑셀 템플릿 생성 기능
- 원본 파일 보호 (결과 파일 자동 생성)

## 설치 방법

### 방법 1: 실행 파일 사용 (권장 - Python 설치 불필요)

1. GitHub에서 `자동검수/dist/AI번역검수도구.exe` 파일을 다운로드
2. 파일을 더블클릭하여 실행
3. 끝! (Python 설치 불필요)

### 방법 2: 소스 코드에서 실행

1. Python 3.8 이상이 설치되어 있어야 합니다.

2. 필요한 패키지 설치:

```bash
cd 자동검수
pip install -r requirements.txt
```

3. 프로그램 실행:

```bash
python gui_app.py
```

### 방법 3: 소스 코드에서 실행 파일 빌드

1. Python 3.8 이상 설치
2. 필요한 패키지 설치:
   ```bash
   cd 자동검수
   pip install -r requirements.txt
   ```
3. 실행 파일 빌드:
   ```bash
   build_exe.bat
   ```
4. `dist` 폴더에 생성된 `AI번역검수도구.exe` 실행

## 사용 방법

1. API 키 입력:

   - Gemini API 키를 입력합니다.
   - "API 키 확인" 버튼을 눌러 유효성을 확인합니다.

2. 언어 선택:

   - 출발언어와 도착언어를 선택합니다.

3. AI 모델 선택:

   - 사용할 Gemini 모델을 선택합니다.
   - "모델 목록 새로고침" 버튼으로 최신 모델 목록을 가져올 수 있습니다.

4. 파일 선택:

   - 검수할 엑셀 파일을 선택합니다.
   - 또는 "엑셀 템플릿 생성" 버튼으로 템플릿을 생성할 수 있습니다.

5. 검수 시작:
   - "검수 시작" 버튼을 클릭합니다.
   - 진행 상황은 실시간으로 표시됩니다.

## 엑셀 파일 형식

### 필수 컬럼

- 출발언어 컬럼 (예: KO, JP, CN 등)
- 도착언어 컬럼 (예: ENG, JP, CN 등)

### 결과 컬럼 (자동 생성)

- **AI 번역**: 개선된 번역 텍스트
- **AI 검수 등급**: 등급 (양호/보통/크리티컬)
- **AI 검수**: 등급 판단 이유
- **일관성 체크**: 일관성 분석
- **수정필요 체크**: 크리티컬 등급인 경우 "✓" 표시

## 결과 파일

검수 결과는 `{원본파일명}_Result.xlsx`로 저장됩니다.
원본 파일은 안전하게 보존됩니다.

## 주의사항

- API 키는 안전하게 보관하세요.
- 대용량 파일의 경우 검수에 시간이 걸릴 수 있습니다.
- API 할당량을 확인하여 토큰 부족 오류를 방지하세요.
- 검수 중에는 파일을 엑셀에서 열지 마세요.

## 문제 해결

### API 키 오류

- API 키가 올바른지 확인하세요.
- Google Cloud Console에서 API 키 상태를 확인하세요.

### 토큰 부족 오류

- API 할당량을 확인하세요.
- 유료 플랜으로 업그레이드하거나 다음 날까지 대기하세요.

### 파일 로드 오류

- 파일이 엑셀에서 열려있지 않은지 확인하세요.
- 파일이 손상되지 않았는지 확인하세요.
